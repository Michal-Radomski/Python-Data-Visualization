**Bokeh is a Python library for creating interactive, web-ready visualizations.** It enables the creation of plots like
scatter, line, and bar charts that users can zoom, pan, and hover over directly in web browsers. Unlike static plotting
tools, Bokeh outputs HTML and JavaScript for high-performance interactivity.[2]

## Ubuntu Installation

Install Bokeh on Ubuntu using pip after ensuring Python 3 and pip are available. Run `sudo apt-get -y install python3-pip`
followed by `pip3 install bokeh` to download the library and its dependencies. Verify the installation with `bokeh --version`
or by importing in Python: `from bokeh.plotting import figure, show`.[4][2]

## Basic Usage Example

Start with a simple script to generate a plot. Create a figure with `figure()`, add glyphs like circles via `p.circle()`,
then display using `show(p)`. For server apps, use `bokeh serve your_script.py` to host interactive dashboards.[7][2]

## JavaScript/TypeScript Equivalent

BokehJS serves as Bokeh's direct equivalent in JavaScript/TypeScript, powering the client-side rendering of Python-generated
plots. Written primarily in TypeScript, it handles model deserialization from JSON and browser-based interactivity, allowing
standalone use in JS/TS projects via npm packages or CDN.[1][5][7]

Create a file named `simple_bokeh_plot.py` with the following Bokeh example code for a basic interactive scatter plot.

```
from bokeh.plotting import figure, show
from bokeh.io import output_file

# Sample data
x = [1, 2, 3, 4, 5]
y = [6, 7, 2, 4, 5]

# Output to HTML file
output_file("scatter_plot.html", title="Simple Bokeh Scatter Plot")

# Create figure
p = figure(title="Scatter Plot Example", x_axis_label="X Values", y_axis_label="Y Values", height=400, width=500)

# Add scatter glyphs
p.circle(x, y, size=10, color="blue", alpha=0.7)

show(p)
```

## How to Run

Ensure your Bokeh venv is activated (`source bokeh_env/bin/activate`). Run `python simple_bokeh_plot.py` in the terminal.
This generates `scatter_plot.html`; open it in a web browser to view the interactive plot with zoom, pan, and hover
tools.[1][3]
