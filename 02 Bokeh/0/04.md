Here's how to embed Bokeh charts in Flask, Django, and React.js apps using the `components()` method for standalone
interactive plots.

## Flask Example

**app.py**

```python
from flask import Flask, render_template
from bokeh.plotting import figure
from bokeh.embed import components

app = Flask(__name__)

@app.route('/')
def index():
    # Create Bokeh plot
    p = figure(title="Embedded Bokeh Chart", height=400, width=800)
    p.line([1, 2, 3, 4, 5], [2, 5, 3, 7, 8], line_width=3)

    # Get script and div components
    script, div = components(p)

    return render_template('index.html', script=script, div=div)

if __name__ == '__main__':
    app.run(debug=True)
```

**templates/index.html**

```html
<!DOCTYPE html>
<html>
  <head>
    {{ bokeh_css|safe }}
  </head>
  <body>
    <h1>Bokeh in Flask</h1>
    <div>{{ div|safe }}</div>
    {{ script|safe }}
  </body>
</html>
```

**Install:** `pip install flask bokeh`

## Django Example

**views.py**

```python
from django.shortcuts import render
from bokeh.plotting import figure
from bokeh.embed import components

def bokeh_chart(request):
    p = figure(title="Django Bokeh", height=400, width=800)
    p.vbar(x=[1, 2, 3], top=[10, 20, 30], width=0.5)

    script, div = components(p)
    return render(request, 'bokeh_chart.html', {'script': script, 'div': div})
```

**templates/bokeh_chart.html**

```html
{% load static %}
<!DOCTYPE html>
<html>
  <head>
    {% load static %} {{ bokeh_css|safe }}
  </head>
  <body>
    <div class="container">
      <h1>Django + Bokeh</h1>
      {{ div|safe }} {{ script|safe }}
    </div>
  </body>
</html>
```

**urls.py**

```python
from django.urls import path
from . import views

urlpatterns = [
    path('chart/', views.bokeh_chart, name='bokeh_chart'),
]
```

**Install:** `pip install django bokeh`

## React.js Example

**App.js**

```jsx
import React, { useEffect, useState } from "react";

function BokehChart() {
  const [script, setScript] = useState("");
  const [div, setDiv] = useState("");

  useEffect(() => {
    // Fetch Bokeh components from Flask/Django API endpoint
    fetch("/api/bokeh-chart")
      .then((res) => res.json())
      .then((data) => {
        setDiv(data.div);
        setScript(data.script);
      });
  }, []);

  return (
    <div className="App">
      <h1>React + Bokeh</h1>
      <div dangerouslySetInnerHTML={{ __html: div }} />
      <div dangerouslySetInnerHTML={{ __html: script }} />
    </div>
  );
}

export default BokehChart;
```

**Flask API endpoint (for React) - flask_api.py**

```python
@app.route('/api/bokeh-chart')
def bokeh_json():
    p = figure(title="React Bokeh", height=400, width=800)
    p.circle([1, 2, 3, 4], [2, 5, 3, 7], size=20)

    script, div = components(p)
    return {'script': script, 'div': div}
```

## Backend API for React (Complete Flask Example)

**Complete flask_api.py**

```python
from flask import Flask, jsonify
from bokeh.plotting import figure
from bokeh.embed import components

app = Flask(__name__)

@app.route('/api/bokeh-chart')
def bokeh_chart_api():
    p = figure(title="Interactive Chart", height=400, width=800, tools='pan,wheel_zoom,reset')
    x = [1, 2, 3, 4, 5, 6]
    y = [2, 5, 3, 7, 8, 4]
    p.line(x, y, line_width=3, line_color='orange')

    script, div = components(p)
    return jsonify({'script': script, 'div': div})

if __name__ == '__main__':
    app.run(debug=True)
```

## Key Points

**Components Method** [geeksforgeeks](https://www.geeksforgeeks.org/python/how-to-create-charts-in-bokeh-with-flask/)

- `components(plot)` returns `(script, div)` tuple
- `div` contains plot container HTML
- `script` contains BokehJS initialization code

**Template Requirements**

- Always include `{{ bokeh_css|safe }}` in `<head>`
- Use `{{ div|safe }}` and `{{ script|safe }}` in body
- `|safe` filter prevents HTML escaping

**React Integration**

- Fetch components as JSON from API endpoint
- Use `dangerouslySetInnerHTML` to render Bokeh HTML
- Components remain fully interactive (zoom, pan, hover)

**Run Examples**

```
# Flask: python app.py → http://localhost:5000
# Django: python manage.py runserver → http://localhost:8000/chart/
# React: npm start (with Flask API running)
```

All examples produce fully interactive Bokeh charts embedded in your web framework with zero server-side Bokeh required after
initial render!
